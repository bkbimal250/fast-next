# SPA Job Portal - Frontend

Location-intelligent, SEO-first SPA Job Portal Frontend built with Next.js App Router.

## Features

- ✅ SEO-optimized pages with SSR
- ✅ Location-based job search
- ✅ Apply without login
- ✅ Message SPA directly
- ✅ Analytics tracking
- ✅ Dynamic city/area pages

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables in `.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:8010
```

3. Run development server:
```bash
npm run dev
```

Visit `http://localhost:3000`
